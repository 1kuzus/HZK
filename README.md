## 说明
`code`文件夹中是在BorlandC+DosBox环境下的调用接口，包括头文件和源文件  
`font`文件夹中是汉字库文件  

|缩写|说明|
|:-:|:-:|
|ASC|ASCII码库|
|HZK|汉字库|
|S|宋体|
|F|仿宋|
|H|黑体|
|K|楷体|
|\_B|Bold 粗体|
|\_T|Traditional 繁体|
